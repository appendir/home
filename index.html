تیم اپند

صفحه ما در github

تیم اپند مجموعه‌ای از فارغ‌التحصیلان متخصص که گرد هم آمده‌اند تا تجربه‌های کاربری بهتری از فناوری برای کاربران خود فراهم آورند

برای مشاهده مطالب و پروژه‌های تیم اپند می‌توانید به اینجا مراجعه کنید.

این صفحه با امکانات GitHub Pages با استفاده از پوسته Cayman theme که توسط Jason Longنوشته‌شده‌است، ایجاد گردیده.
